# Portfolio

## Overview
A React-based portfolio website built with Create React App.

## Project Structure
- `public/` - Static assets and HTML template
  - `img/` - Images including cryptocurrency SVGs
- `src/` - Source code
  - `components/` - React components
    - `Footer/` - Footer component
    - `Pages/` - Page components (Home, Contact)
    - `Success.js` - Success page component
  - `App.js` - Main App component with routing
  - `index.js` - Entry point
  - `index.css` - Global styles

## Tech Stack
- React 18
- React Router DOM v6
- Create React App (react-scripts)

## Running the Application
The app runs on port 5000 in development mode.

## Environment Variables (Development)
- `PORT=5000` - Server port
- `HOST=0.0.0.0` - Host binding
- `DANGEROUSLY_DISABLE_HOST_CHECK=true` - Allow all hosts for Replit proxy
