# 💱 Daze-T Frontend (Crypto → M-Pesa)

React frontend for the **Daze-T Crypto Payment Platform**.

This application allows users to:

- Convert USD to KES in real-time
- Select a cryptocurrency
- Enter an M-Pesa phone number
- Generate a crypto payment invoice
- Automatically redirect to a secure payment page

This frontend connects to the Daze-T Flask backend API.

- ✅ Phone number validation (Kenyan format)
- ✅ Dark mode support
